### 问题 2：资金面维度不全（北向5日净流/ETF5日申赎缺失）

#### 当前表现
- **UI 显示**：
  ```
  资金数据状态：部分缺失
  
  缺失维度：北向5日净流、ETF 5日申赎
  ```
- **API 响应**：`capital_game_summary.northbound` 显示 `net_inflow_5d: null`
- **用户困扰**："为什么每次都缺这两项？它们能补全吗？"

#### 根本原因

| 维度 | 来源 | 问题 |
|------|------|------|
| 北向5日净流 | akshare (`stock_hsgt_individual_em`) | ✅ 有API，但需要 bypassproxy + akshare 库 |
| ETF5日申赎 | 东方财富逐股 ETF 流向 | ❌ 无开放 API；需要爬虫或数据市场 |

#### 改进方案

**短期（优先）**：增强缺失维度的说明和代理估算
```python

            "proxy_value": -15883311104.0,  # 从融资融券推估

            "reason": "无开放 API，需付费数据源",

        "status": "ok",
        "net_inflow_5d": null,  # 原始数据
```

## 结论优化的严格标准

### 必填观点清单

#### 1. 技术面（必须）
- [ ] 当前价格与 MA5/MA20 的关系（上方/下方/接近）
- [ ] 趋势判断（偏多/偏空/震荡）
- [ ] 近期波动特征（加速/减速/整理）
- [ ] MACD/KDJ 状态（金叉/死叉/超买/超卖）

#### 2. 资金面（必须）
- [ ] 5 日/1 日主力资金净流（流入/流出/中性）
- [ ] 龙虎榜近 30 日上榜频率和净买入方向
- [ ] 融资融券 5 日变化（资金面是否宽松）
- [ ] 缺失维度的说明（北向/ETF 为什么缺）

#### 3. 消息面（如有）
- [ ] 近期新闻/政策数量和质量
- [ ] 是否存在重大事件催化
- [ ] 社交热度是否异常

#### 4. 预测可靠性（必须）
- [ ] 7 日准确率和样本数
- [ ] 信号是否可执行（样本 ≥5、覆盖率 ≥5%）
- [ ] 如果不可执行，应该如何处理（降仓/观望）

#### 5. 操作建议（必须）
- [ ] 当前建议（加仓/持有/减仓）
- [ ] 触发条件（价格位置/信号变化）
- [ ] 止损条件（明确的下行界限）
- [ ] 加仓条件（明确的上行机会）
