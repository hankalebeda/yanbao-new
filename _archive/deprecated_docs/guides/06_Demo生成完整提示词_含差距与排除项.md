【任务】根据项目文档，生成/补全 A 股个股研报平台的**前台 Demo**，纯前端展示，使用 Mock 数据，不需要后端接口。

【目标】
1. 让你在写正式代码前，就能看到系统长什么样
2. 清晰展示每个页面的模块结构、布局和每个模块的数据内容

- 纯 HTML + CSS + 少量 JS（无需框架）
- 所有数据用 Mock，可硬编码
- 不连接真实 API

- 页面 375px～桌面端无横向溢出
